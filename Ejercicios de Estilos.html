Temas básicos de HTML y CSS

1. Creación de un portal básico en HTML

HTML es el lenguaje estándar para crear estructuras de páginas web. Consiste en elementos que definen el contenido y la estructura del sitio.

Sintaxis:


    
Bienvenidos a mi portal

    
Este es un ejemplo básico de HTML didactico.

Uso:

Definir encabezados, párrafos y la estructura inicial de una página.

2. Sintaxis básica de funcionalidades CSS

CSS (Cascading Style Sheets) define el estilo y diseño de una página web. Permite separar el contenido de la presentación.

Sintaxis:

selector {
    property: value;
}

Ejemplo:

body {
    background-color: #f0f0f0;
    font-family: Arial, sans-serif;
}
